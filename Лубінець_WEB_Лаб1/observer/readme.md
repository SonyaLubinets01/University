   # Observer (Поведінковий шаблон)

## Опис проблеми
Необхідно реалізувати механізм, при якому зміна стану одного об’єкта
автоматично повідомляє всі залежні об’єкти без жорсткого зв’язку між ними.

---

## Ідея реалізації
Шаблон Observer реалізує залежність типу "один-до-багатьох".
Об’єкти-спостерігачі підписуються на об’єкт-джерело та отримують
сповіщення про всі зміни його стану.

---

## Де застосовується
- графічні інтерфейси користувача;
- системи сповіщень;
- подієві системи;
